# 🎬 StreamVerse

StreamVerse is a full-stack movie and streaming platform built using HTML, CSS, JavaScript, Node.js, and Express.js.

The project allows users to browse movies across multiple categories, create accounts, log in, search content, and watch available videos through a simple streaming interface.

## 🚀 Features

### Authentication System

* User Signup
* User Login
* Guest Login
* Session Management
* Password Hashing using SHA-256

### Movie Platform

* Browse Movies
* Search Movies
* Category Filtering
* Trending Section
* Guest and Member Access Levels
* Video Streaming Support

## 🔐 Authentication

Users can:

* Create an account
* Login securely
* Continue as a guest

Member accounts unlock all available content while guest users have limited access.

## ▶️ Run Locally

Install dependencies:

```bash
npm install express cors
```

Start the server:

```bash
node Server.js
```

Open:

```text
http://localhost:3000
```
